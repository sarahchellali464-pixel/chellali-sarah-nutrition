
 Module Nutrition

Nom : Chelali Sarah

Recette saine : Salade équilibrée

Ingrédients :
- 1 tomate
- 1 concombre
- 1 avocat
- Jus de citron
- 1 cuillère à soupe d’huile d’olive

Préparation :
1. Laver les légumes
2. Couper la tomate, le concombre et l’avocat
3. Mélanger dans un bol
4. Ajouter le citron et l’huile d’olive

Bienfaits :
Cette recette est riche en vitamines et bonne pour la santé.
